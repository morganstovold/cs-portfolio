Online Texas Holdem

Link

This online Texas Hold’em poker game offers a real-time, interactive experience for players. The frontend, developed with Next.js, ensures fast, responsive gameplay with smooth animations and a user-friendly interface. On the backend, I used Rust, which brings the performance and security needed for real-time multiplayer interactions. The game supports features like in-game chat, player avatars, and a live leaderboard, creating a highly engaging environment. Designed to scale efficiently, it can handle multiple simultaneous games and players, ensuring a seamless experience for poker enthusiasts of all skill levels.

Snap Drawing App

Link

The Snap Drawing App is a lightweight, easy-to-use platform for creating digital sketches. Whether for quick doodles or detailed illustrations, the app offers a wide range of tools, including different brush styles, colors, and layer options. It supports real-time collaboration, allowing multiple users to draw on the same canvas simultaneously. Built for both desktop and mobile devices, the app features a clean, intuitive interface, ensuring a smooth user experience. With save/export functionality, undo/redo options, and customizable brush settings, it caters to both casual users and professional artists.
